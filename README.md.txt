# 📊 리뷰 분석 AI 데모 서비스

CSV 리뷰 데이터를 업로드하면  
AI(GPT)가 자동으로 감성 분석, 종합 만족도 평가, 요약 리포트를 생성하는  
**Streamlit 기반 데모 웹 서비스**입니다.

---

## ✨ 주요 기능

- 📂 CSV 파일 업로드 (`review` 컬럼 필수)
- 🤖 AI 기반 리뷰 감성 분석
- ⭐ 종합 만족도 점수 (10점 만점)
- 📊 감성 분포 시각화 (막대 / 파이 차트)
- 🔑 주요 키워드 추출
- 📝 AI 요약 리포트 생성
